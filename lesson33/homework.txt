ЗАДАНИЕ 1
Написать программу для получения погоды в разных городах. 
Пользователь вводит название одного или нескольких городов в консоль, в ответ программа
печатает на экран таблицу, где первая колонка - название города, вторая - текущая погода, 
третья - описание, четвёртая - влажность
Пример вывода:
city   |   temp   |   description   |   humidity   |
London |   10     |      sunny      |     35       |
Paris  |   12     |       cold      |     36       |

Для красивого вывода таблицы в консоли использовать любую бибилотеку - например prettytable.
Для получения данных использовать сайт openweathermap.org
Для реализации http-клиента использовать бибилотеку requests или свой клиент, написанный с помощью
библиотеки socket.

Для этого:
1. Зарегистрироваться на сайте
2. Перейти во вкладку api keys: https://home.openweathermap.org/api_keys 
3. Скопировать свой ключ.
4. Перейти на страницу документации API: https://openweathermap.org/current#name
5. Тут описан URL и параметры, которые надо отправить по http-протоколу на сервере для получения
информации.
