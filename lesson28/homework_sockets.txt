ЗАДАНИЕ 1
Написать простой однопоточный, синхронный echo-сервер с помощью библиотеки socket. 
Сервер запускается на любом порте (например 5000), и начинает ждать входящее соединение. 
Когда клиент отправляет любую строку на этот сервер, в ответ должна прийти такая же строка 
(поэтому такой сервер и называется эхо-сервером).
Для этого надо сначала посмотреть презентацию и изучить методы интерфейса сокетов протокола TCP.


ЗАДАНИЕ 2
Создать сервер и клиент с помощью библиотеки socket, используя интерфейсы протокола TCP.
Задача сервера: сервер принимает подключение и ждёт пока клиент отправит данные. Клиент отправляет какой-то текст.
Сервер должен вернуть самое встречающееся слово в этом тексте.
Например клиент отправил: Hello python community! My name is Max. I have been working as a python developer for 4 years.
В ответ должно прийти: python (Так как это самое распространённое слово в тексте. Если несколько слов с 
одинаковым показателем встречаемости, то вернуть любое из них).
